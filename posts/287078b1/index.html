距离2024年倒计时1个月

发表于 2023-11-30 分类于 人生录 本文字数： 661 阅读时长 ≈ 2 分钟

the git up —Blanco Brown 很奇妙欢快的一首歌

距离2024年只剩下1个月的时间。最近发现一个规律，随着人年岁的增长，一年的时间会不断被压缩，变得越来越短。快乐的记忆会越来越少，也会越来越珍贵。

10岁的我，80%的日子都很有趣。有初春的风，送风筝上青天。有清明的油菜花，和暖阳下蜂蜜共舞。有夏至的雨，不打招呼自来带来夏天的讯息。还有冬天的厚雪，铺满竹林和草地。

15岁的我，50%的日子都很有趣。有考试结束后畅快看电视的自由，有冬日火炉旁的冒着腾腾热气的红薯，有荣誉和艳羡，有骄傲和好强的心气。

18岁的我，30%的日子都很有趣。依旧能够抬眼看到碧蓝的天，可以跨越2千千里的距离来到临海城市开始新的生活。有迷茫和坚定选择的权利，有笑声和平等的生活状态。

23岁的我，10%的日子还算有趣。可以挣钱，可以买到自己期盼已久的东西。终于有了选择权和话语权，人人都不敢惹我，我好像成了一个成熟的大人。

25岁的我，不太知道还有多少日子会有趣。只明白，时间会带走一切，但闪光，浪漫的记忆会永远在，若当日又用满含热情的文字记录下来，那便是我死亡前一瞬回马灯般闪现的全部。

所以一定不要停笔，要写，要挣扎。在能大笑的时候笑出最大声，在能哭的时候哭出最大声，在能入睡的每个夜晚酣然入睡。始终要浪漫，潇洒，投入的活出每一个值得珍惜的瞬间，这些才是最重要的。

剩下的：摧残人心的日子，艰苦卓绝的时刻，焦虑万分，不能入睡的每个夜晚，每个不属于死亡的日子，都去他妈的！抽出灵魂，不去思索，做一个木头，行动行动行动，等着这些难熬的日子过去。

看，又是一个艳阳天。

突然发现，我只活在两个地方，在当下一刻，和脑海中还留存下的记忆。留存的记忆多为琐碎模糊的片刻，当下又被情绪填满。

这样，活的真没价值呀，全是情绪，还是无用的，不好的情绪居多。
