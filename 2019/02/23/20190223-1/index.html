    至关重要的是现金流，而不是盈利

    • Pat Dorsey

      身份：作家

      资历：西北大学毕业，晨星股票研究部负责人

      观点：价值投资理论

      作品：《股市真规则》《巴菲特的护城河》

    • Joe Mansueto

      身份：晨星创始人

      资历：University of Chicago (BA) (MBA)

      观点：

      作品：晨星公司

    核心主题

    • 建立投资组合核心三原则

      1. 找到正确的公司（竞争优势）
      2. 知道何时买入卖出（安全边际）
      3. 长期持有

    • 至关重要的是现金流，而不是盈利。

    • 竞争优势《巴菲特的护城河》

      1. 历史上盈利情况
      2. 获得巨额利润的原因
      3. 能够持久抵御竞争者的时间
      4. 行业竞争情况

    • 投资语言

    • 公司分析基础

    摘抄

    • 在投资方面没有很多巨著，你应当有能力掌握它们中的大部分。
    • 至关重要的是现金流，而不是盈利
    • 如果你无法知道1美元现金是怎样从一家公司的消费者转到股东手里，肯定是某些地方出了问题。要么是公司的商业模式太混乱，要么是你还需要更深的挖掘和研究。
